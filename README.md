# 📊 PwC Digital Accelerator - Power BI Project  

### 🚀 Created by [Kunal Suhanda]  

This project is part of the **PwC Digital Accelerator Job Simulation**, where I analyzed business datasets and built **interactive Power BI dashboards** to derive meaningful insights.  

## 📌 Project Overview  
The project includes **three business cases**, each requiring data analysis and visualization in **Power BI**:  

1. **📞 Call Center Trends:** Analyzing call center data to optimize performance.  
2. **🔄 Customer Retention:** Identifying churn risk and retention strategies.  
3. **🌍 Diversity & Inclusion:** Understanding gender balance in executive roles.

## **📊 Dashboards & Insights**  

### **1️⃣ Call Center Trends Dashboard**  
**Objective:** Analyze call center performance and customer service efficiency.  

📌 **Key Insights:**  
✅ **Average Satisfaction Rating:** `3.4 / 5`  
✅ **Total Calls Answered:** `4,050` | **Abandoned:** `950`  
✅ **Average Speed to Answer:** `67.52 seconds`

### **2️⃣ Customer Retention Dashboard**  
**Objective:** Identify churn risk and strategies to retain customers.  

📌 **Key Insights:**  
✅ **Churn Rate:** `27%`  
✅ **High Risk Factors:** `Month-to-month contracts`, `high monthly charges`  
✅ **Retention Strategy:** Increase `yearly contracts` & `automatic payments`

### **3️⃣ Diversity & Inclusion Dashboard**
